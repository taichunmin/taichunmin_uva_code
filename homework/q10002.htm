Q10002: Center of Masses

找出一個凸多邊形（convex polygon）的重心。

Input

一組測試資料代表一個凸多邊形，第一個數字 n代表這個多邊形有 n 個頂點(n<= 100) 。接下來會有 n 對整數為這 n 個頂點的座標。給定的座標並沒有照順序。如果測資的多邊形為不合理的，也就是給定的 n<3 ，那代表檔案結束，並且不要執行這組測資。

Output

輸出每個凸多邊形重心的座標，請以小數表示，並且小數點後面要取3位。

Sample Input

4
0 1
1 1
0 0
1 0
3
1 2
1 0
0 0
7
-4 -4
-6 -3
-4 -10
-7 -12
-9 -8
-3 -6
-8 -3
1

Sample Output

0.500 0.500
0.667 0.667
-6.102 -7.089
