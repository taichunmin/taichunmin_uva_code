Q10830: A New Function

我們都知道所有整數 N 都可以被 1 和 N 整除。這就是為何這兩個數永遠不會被視為任何一個數N的因數。

函數 SOD(n) 被定義為所有 n 的因數之和。舉例來說 SOD(24) = 2+3+4+6+8+12 = 35。

而函數CSOD被定義如下：

給你 n 的值，你必須找出 CSOD(n) 的值。

Input

輸入檔案最多包含50列。每一列含一個整數 n（0 <= n <= 2000000000）。

當 n=0 時代表輸入結束。 

Output

每一輸入請輸出一列。這一列包括這是第幾組測試資料以及 CSOD(n) 的值。

你可以放心答案必定可以由 64-bit 的有號整數(64-bit signed int)表示。
