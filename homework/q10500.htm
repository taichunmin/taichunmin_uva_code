Q10500: Robot maps

The Problem

ACM工廠生產很多危險的物品並且放在工廠裡。這就是為什麼沒有人被允許進入儲藏室，只有特殊的機器人才可以進去儲存新貨物或者提取貨物來賣。ACM工廠正在測試一種機器人，這種機器人能建造地圖並記錄它走過的地方。

這個機器人一剛開始可以被放在儲藏室的任何一個地方。在開始移動前，機器人把地圖切成格子狀，並把每一個格子初始化成‘?’，代表「未知的格子」。機器人剛開始的位置一定是空的，也就是說這個地方沒有物品。首先，機器人把起點記錄在地圖上。然後，機器人探察四個方向的感應器的訊號，這四個感應器分別回傳機器人的北方、東方、南方及西方這四個格子是不是可以走的。如果不能走，感應器會回傳‘X’，否則感應器會回傳‘0’代表可以走，並且機器人會把這四個格子的狀態會記錄到地圖上。然後，機器人會選擇第一個可以走的格子（註1）前進，並且重覆這個動作 ──偵測身邊四個格子的狀態，並且前進到下一個可以走的格子。當機器人的四個方向皆為物品或者探察過了，就會停下來。最後，機器人會把儲藏室的地圖印出來。

註1：檢查可以走的格子的順序，首先是北方，然後順時針檢查。檢查到第一個可以走的格子就前進。

註2：輸入和輸出的‘0’為數字「零」。

Input

每一組測資包涵 2+N 行。第一行包涵兩個整數，N 和 M，代表這個地圖的大小。兩個整數用一個空白隔開，而且符合  0 < N <= 10，0 < M <= 10。第二行有兩個整數，x_ini  和 y_ini機器人開始的位置。兩個整數用一個空白隔開，而且符合1 <= x_ini  <= N 和 1 <= y_ini <= M。接下來的 N 行，每行有 M 個字母，‘X’ 或者是 ‘0’。字母和字母之間有一個空白相隔。‘X’代表有物品，‘0’代表空的區域。這個地圖一定是正確的，也就是說機器人開始的位置一定是空的區域。當 N=0 且 M=0 時代表輸入的結束。

Output

每一組測資都要先印出一個空白行，接下來是機器人偵測到的地圖，再一個空白行，再來是機器人移動了幾步。

機器人偵測到的地圖格式如下：

| s | s | … M … | s |
|---|---| … M … |---|
| s | s | … M … | s |
|---|---| … M … |---|        s Î {‘X’, ‘0’, ‘?’}
… N …
|---|---| … M … |---|
| s | s | … M … | s |
|---|---| … M … |---|
