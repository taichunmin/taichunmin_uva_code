Q10919: Prerequisities?

大學一年級新生 Freddie決定要選 k 門課程，為了符合該學位的規定，他必須在各個類別的課中選一些課，根據他所選的課，你能確定Freddie可以畢業嗎？

Input

輸入包含很多組測資，每組測試資料的第一列包含 Freddie所選的課程數 k ( 1 <= k <= 100 ) 和 課程類別的數目 M ( 0 <= M <= 100 )。接下來的一或多列包含了 k 個 課程代碼號（每個代碼含4碼數字），這些就是Freddie所選的課程。對每一個課程類別包含了該類別所有課程的數量 c ( 1 <= c <= 100 ) 及必修的數量 r ( 1 <= r <= c )，再來是 c 個課程代碼，同一種課程代碼可能同時存在幾個課程類別中。

Freddie選的課不會有重複的。各類別中的課也不會有重複的。

最後一列單獨一個 0 代表輸入結束。

Output

如果Freddie所選的課程，滿足每一個類別，他就達到畢業標準，所以輸出"yes"，反之，輸出"no"。
