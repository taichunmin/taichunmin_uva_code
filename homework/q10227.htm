Q10227: Forests

「如果一棵樹在森林中倒下，但是在那裡沒有人聽到，那這樣有發出聲音嗎？」這句經典的文字是由一位愛爾蘭哲學家 George Berkeley所創造的。

在本問題中，跟森林中的樹以及人有關。有一個森林有T棵樹（編號從1到T）和P個人（編號從1到P）。

Input

輸入的第一列有一個整數代表以下有幾組測試資料。

每組測試資料的第1列包含2個整數 P 和 T（ 0 <= P,T < 100），接下來有多列輸入，每列有2個整數 i 和 j。表示編號 i 的人聽到編號 j 的樹倒下。不同的人可能聽到不同的樹倒下。每組測試資料以一列空白列結束（最後一組測試資料可能以檔案結束符號結束）。

輸入的第一列與第一組測試資料間有一空白列。各組測試資料間亦有一空白列。請參考Sample Input。

Output

如果有2個人所聽到的倒下的樹完全一樣，我們說這2個人有相同的觀點（例如第一組測試資料中的第1、第3個人都聽到第2、第3棵樹倒下）。對每組測試資料輸出一列，這P個人中共有多少種不同的觀點。。測試資料間請空一列。 請參考Sample Output。
