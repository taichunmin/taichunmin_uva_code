Q10374: Election

加拿大是個多黨政治的社會。一般的候選人會隸屬於某一政黨，而贏得最多席次的政黨就成為執政黨。但是也有一些候選人士獨立參選的，他們不隸屬於任一政黨。你的任務就是計算某一選區的選票，然後回答是哪一個政黨贏得該選區的席次。

Input

輸入的第一列有一個整數，代表接下來有多少組測試資料。接下來有一空白列，並且在測試資料間亦有一空白列。

每組測試資料的第一列有1個正整數 n（2 <= n <= 20），代表該選區有多少名候選人。接下來的2*n列為這些候選人的資料，每一候選人2列，第一列為候選人的名字，第二列為該候選人的政黨名稱（如果是獨立參選候選人，此位置以"independent"來取代）。沒有候選人的名字會重複，也沒有政黨的名稱會重複。每列的長度最多80個字元，而且每列的最前面及最後面都不會有沒有意義的空白字元出現。

接下來的一列有一個正整數m（m <= 10000），代表共有m張選票要計算。接下來的m列每列代表一張選票，內容為某候選人的名字。如果這名字不在候選人的名單中，那這張選票應被視為廢票，不列入計算。

Output

每組測試資料應輸出以下其中一種：

  • 如果有某一位隸屬於某政黨的候選人贏得該選區的選舉，請輸出該政黨的名稱。
  • 如果是獨立參選的候選人贏得選舉，請輸出"independent"
  • 如果沒有任一候選人得到的選票比其他候選人多的話，請輸出"tie"

測試資料間輸出亦請空一列，請參考Sample Output。
