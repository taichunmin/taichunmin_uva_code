Q10081: Tight Words

給你一個數字 k（0 <= k <= 9），我們說一個由 0~k 組成且長度為 n 的字串是「緊密」的，如果任何相鄰的數字相差不會超過 1 的話。

例如：k=2, n=3, 那所有可能的字串為 000, 001, 002, 010, 011, 012, 020, 021, 022, 100, 101, 102, ......, 220,221,222。共27個。

其中 000, 001, 010, 011, 012 這些字串都是「緊密」的。但是 002, 020, 102 這些字串則不是（因為 0 跟 2 相鄰，其差超過 1）。

Input

輸入含有多組測試資料。每組測試資料一列，含有 2 個正整數 k 和 n （ 1 <= n <= 100）。

Output

對每一組測試資料輸出一列，輸出所有可能字串中，「緊密」字串所佔的百分比。請四捨五入到小數點後 5 位。

例如：k=2, n=5, 所有可能的字串共有 243 個，其中「緊密」的字串共有 99 個，所以百分比為 40.74074。請參考Sample Output。

Sample Input Sample Output 
4 1
2 5
3 5
8 7
 
100.00000
40.74074
17.38281
0.10130
