Q11033 Help My Brother

Tuhin正在用一個 4x4 格的板子玩遊戲。他身邊有很多張卡片，每張卡片都有一個數字 1~7。他可以把這些卡片放在還沒放卡片的格子上。因為他對於數學非常擅長，他可以把16張卡片放在這板子上，使得每一行、每一列及每一條對角線的總和都等於N。

因為他是我弟，他有時會來找我幫忙。他帶著他的板子還有卡片來找我。我看他非常困惑而且好像很累。他說，「我給你三個數字 N, M 和 P，你必需把這些卡片放在板子上，像我這樣做。而且如果板子上這16張卡片的數字全部乘起來，除以 M ，餘數不能超過 P 。」當我在想這個問題時他又說了，「我想要知道總共有幾種擺法，還有最開頭的兩種排法是什麼。如果只有一種擺法，那你就告訴我那一種擺法就可以了。求求你寫給個程式給我吧，我要自己去驗證這個題目。」

我明天剛剛好要考試。所以，我現在不能幫我弟，有人可以提供程式給我嗎？先謝謝你了。

Input

檔案中有很多組測資，每組測資有三個整數，N, M ( 0 < M < 100000 ) 和 P ( 0 <= P < M ) ，數子和數字間有空白隔開。

檔案最後以 N = M = P = 0 當作結束。同時，這個測資不需要被執行。

Output

每一組答案，你必需先印出他是第幾組，從第1組開始計數。
第二行印出有幾種解法。
如果解法有兩種以上，那麼你就把前兩種解法印出來。
第一種解法的意思是說如果你把第二列、第三列和第四列放到第一列的後面，這個組合起來的數字最小的就是第一種解法。
當你印出一個盤面後，請再印出4個點(`.')當作分隔。
