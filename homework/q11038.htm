Q11038: How many 0's?

一位僧侶用十進位寫下了 m 和 n 之間所有的整數，0 ≦ m ≦ n，他總共需要寫下多少個「0」？

Input

每組測試資料一列，每列有兩個無號32-bit整數 m 及 n，且m≦n。
一列含兩個負數代表輸入結束。

Output

對於每一組測試資料，輸出此代表僧侶需寫下多少個 0 。

Sample Input

10 11
100 200
0 500
1234567890 2345678901
0 4294967295
-1 -1

Sample Output

1
22
92
987654304
3825876150
