Q10493: Cats, with or without Hats

一隻貓會戴一頂帽子只有在有 N 隻貓在他的帽子裡的情況下，反之亦然。而且，僅有一隻貓不在其他貓的帽子裡。假如有 M 隻貓沒有戴帽子，請問總共有多少隻貓？

　

Input

輸入含有多組測試資料。

每組測試資料一列，有 2 個整數 N 和 M（1 <= N < 100000,  1 <= M <= 100000）。

當 N=0 時代表輸入結束。請參考Sample Input。

Output

對每一組測試資料輸出一列，首先輸出 N M，然後假如所有貓的數目是唯一的答案，則輸出這個整數。如果情況不可能請輸出 Impossible。如果有不只一個答案，請輸出 Multiple。

輸出格式請參考 Sample Output。

Sample Input Sample Output 
2 5
3 4
3 3
0 0
 
2 5 9
3 4 Impossible
3 3 4
