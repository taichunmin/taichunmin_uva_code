Q10856: Recover Factorial

階乘（factorial numbers）可以用一連串的質數相乘來表達。例如：4! 可以表達如下：

4! = 2 x 2 x 2 x 3 (質因數的個數： 4)

給你 N ，X! 質因數的個數，你必須找出 X 最小可能為多少。

Input

輸入含有多組測試資料（不會超過1000組）。

每組測試資料一列，有一個整數 N（0 <= N <= 10000001）

當 N<0 代表輸入結束。請參考Sample Input。

Output

對每一組測試資料輸出一列。首先輸出這是第幾組測試資料，然後輸出 X!。如果沒有答案，則輸出 Not possible.

輸出格式請參考Sample Output。

Sample Input Sample Output 
4
240
241
-1
 
Case 1: 4!
Case 2: 101!
Case 3: Not possible.
