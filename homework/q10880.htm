Q10880: Colin and Ryan

Colin 和 Ryan 辦了一場派對。他們共烤了 C 個小餅乾，邀請了 G 個客人，每個客人吃了 Q 個小餅乾，最後剩下 R 個小餅乾 ( R < Q )。

Input

輸入的第一列有一個整數 N，代表以下有多少組測試資料。每組測試資料一列，含有2個整數 C 和 R。( 0 <= R <= C <= 2*109 )

Output

對每組測試資料輸出一列。首先輸出這是第幾組測試資料，接著輸出所有可能的Q值，如果 C = R 則輸出 0 。若答案不只一個，則數字由小到大排序，數字之間以空白間隔。

輸出格式請參考Sample Output，注意每列最後不可以有多餘的空白。

Sample Input Sample Output 
4
10 0
13 2
300 98
1000 997

Case #2: 11
Case #3: 101 202
