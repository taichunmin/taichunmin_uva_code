Q10328: Coin Toss

擲硬幣常常被用來決定一件事情。通常會使用一枚公平的硬幣，就是兩面（H 和 T）出現的機率一樣。在這個問題中硬幣要被擲 n 次，其結果可以用一字串來表示。例如： n=3, 則有 8 種可能的情況：

HHH HHT HTH HTT THH THT TTH TTT

我們有興趣的是 H 連續出現的情況，在上面的情況中，3 個 H 連續的有 1 種，2 個 H 連續的有 3 種，至少有 1 個 H 的有 7 種。 假設一枚硬幣被擲 n 次，可能的結果有 2n 種，這其中有多少種含有連續至少 k 個 H。

Input

輸入含有多組測試資料。每組測試資料一列，含有 2 個正整數 n 和 k （ 1 <= k <= n <= 100）。

Output

對每一組測試資料輸出一列，輸出有多少種含有連續至少 k 個 H。
