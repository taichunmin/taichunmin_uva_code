Q10759: Dice Throwing

投擲 n 個一般的骰子（每個骰子有6面，分別有點數1到6點），點數總和至少 x 的機率是多少？

Input

輸入含有多組測試資料。每組測試資料一列，有 2 個正整數 n （1 <= n <= 24）和 x（0 <= x < 150）。當 n=x=0 時代表輸入結束。

Output

對每一組測試資料輸出一列，輸出點數總和至少 x 的機率是多少（請約分到最簡分數）。所有的數字均可以 64-bit 的整數表達。請參考Sample Output。
