Q11012: Cosmic Cabbages

科學家已經找出在太空中種植甘藍菜的方法。他們建立了一個巨大的3D鋼管方格結構，然後在鋼管交接的地方種1棵甘藍菜。每個鋼管交接處均給予一笛卡兒座標（Cartesian coordinate）。有一隻太空螞蟻想要沿著鋼管從某棵甘藍菜爬到另一棵甘藍菜。太空螞蟻總是選擇最短的路徑，這個路徑的長度稱為這2棵甘藍菜之間的太空距離。給你一堆甘藍菜的座標，請你找出任2棵甘藍菜最大的太空距離。

Input

輸入的第一列有一個整數代表以下有多少組測試資料。每組測試資料的第一列有一個整數 n（2 <= n <= 105），接下來的 n 列，每列有3個整數 各代表一棵甘藍菜的座標（整數範圍為 [ -108 ~ 108 ]。

Output

每組測試資料輸出一列，輸出任2棵甘藍菜最大的太空距離。輸出格式請參考 Sample Output。
