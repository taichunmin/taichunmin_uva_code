Q10784: Diagonal

n 凸多邊形對角線的數目不小於 N，請問  n 最小可能為多少？下圖為 n=6 的凸多邊形對角線的情形（實線）。

Input

輸入包含多組測試資料（小於 1001 組）, 每一列包含一個正整數 N （N <= 1015）。

當 N=0 時代表輸入結束。

Output

對每一組輸入，輸出這是第幾組測試資料以及 n 最小為多少，使得 n 凸多邊形對角線的數目不小於 N。

輸出格式請參考 Sample Output。

Sample Input Sample Output 
10
100
1000
0
 
Case 1: 7
Case 2: 16
Case 3: 47
