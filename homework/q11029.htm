Q11029: Leading and Trailing

所有在程式設計已經有點經驗的人都知道，當 k 很大時你無法完整的表達出 nk。例如： C 語言的函數 pow(123456,455) 能夠用 double 資料型態來表達，但是你卻無法得到所有正確的數字。然而，若是能知道一些最左邊（leading）和最右邊（trailing）數字的話，也可稍微得到一些滿足。

Input

輸入的第一列有一個整數 T（T < 1001），代表有幾組測試資料。接下來的 T 列，每列有2個正整數 n 和 k。n 可以用32位元的整數表達，而 k<10000001。

Output

每組測試資料輸出一列，輸出 LLL...TTT 的樣式。其中 LLL 代表 nk 的最左邊3個數字，TTT 代表 nk 的最右邊3個數字。例如 1234562 = 15241383936，所以你應該輸出 152...936。

你可以假設 nk 至少有6位數。

Sample Input Sample Output 
3
123456 1
123456 2
2100000056 67333
 
123...456
152...936
982...016
